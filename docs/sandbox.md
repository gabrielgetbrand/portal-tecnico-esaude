# Sandbox (Testes)

Disponibilizamos um ambiente de testes previsível, com dados sintéticos e cenários padronizados para validação inicial.

## Acesso e Credenciais
* Como solicitar acesso e obter credenciais/certificados.
* Limites do sandbox: volume, rate limit, janelas de indisponibilidade planejada.

## Dados e Cenários
* Dados sintéticos (não reidentificáveis) e cenários de referência.
* Checklist de smoke tests e critérios para avançar para homologação.
* Status do sandbox e incidentes recentes.
